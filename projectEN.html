Web Design Projects

Stardew Valley Simulation →

This project is a farm simulation game inspired by Stardew Valley and designed using p5.js. The game allows players to perform various farming activities, such as planting seeds, raising livestock, and selling crops and animal products. Advanced features like voice recognition have been integrated, showcasing the innovative application of JavaScript combined with p5.js to deliver a dynamic and interactive gaming experience.

Tetris Simulation →

This project is a creative reimagination of the classic game Tetris, developed using the p5.js JavaScript library. It aims to evoke childhood nostalgia while teaching players how to effectively manage resources and balance competing interests. By leveraging p5.js, the game integrates interactive elements and dynamic visual effects, enhancing the traditional gaming experience with modern web functionalities. The design focuses not only on entertainment but also on fostering practical skills through gameplay.

Flappy Bird →

This project is a creative adaptation of the popular game Flappy Bird, incorporating advanced features like motion and color tracking. In this version, players can use motion tracking technology to control the bird's ascent or descent with their left or right hand. Additionally, an innovative "Change Color" button allows players to alter the bird's color by capturing the hue from a selected area in the game environment, adding a unique interactive element to the gameplay.

3D Project →

This is an innovative application developed using p5.js, designed to create a dynamic 3D home model. The project leverages the 3D rendering capabilities of the p5.js library, allowing users to interactively explore and visualize architectural designs from various perspectives. Users can navigate the virtual space, experience the layout and design features of the home, and enjoy a highly immersive environment.

Interactive Chatroom Platform →

This involves creating an interactive chatroom platform that enables real-time communication. Developed using PHP, the platform allows users to create their own accounts and join live chatrooms where they can interact with others. Key features include user authentication, real-time messaging, and customizable user profiles, making it an engaging and versatile tool for online communication.

Simpson Character Test →

This is an engaging character-matching quiz developed using PHP. Based on user responses to a series of questions, the tool connects users with characters from the popular TV show The Simpsons. This interactive platform analyzes user input to determine which Simpson character they resemble the most, delivering a fun and personalized experience. Additionally, it features a unique option to view frequency statistics for different character matches, displaying these results as percentages for added insight and enjoyment.
